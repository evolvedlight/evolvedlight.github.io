title: Django quickstart with Ubuntu
---

The Django tutorial is great, but we don't need quite the same about of detail if we just want to use Django in a simple way on Ubuntu.

First, install python-pip:

`sudo apt-get install python-pip`

Next, install Django using pip. As the tutorial suggests, doing this in a virtualenv provides some benefits - but since this is a simplied tutorial, we'll just install it normally:

`sudo pip install django`

Great! All installed.
